#To-Do List App

A simple command-line to-do list application where you can add, view, complete, and delete tasks. Tasks are saved in a `tasks.json` file so they persist between runs.

### Features
- Add new tasks
- Mark tasks as done
- Delete tasks
- View all tasks
- Saves tasks to `tasks.json`

### ▶️ How to Run
```bash
python todo.py
